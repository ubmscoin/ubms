UBMSCOIN (UBMS) 백서 v1.0
1. 개요
**UBMSCOIN (UBMS)**은 완전히 새로 설계된 차세대 블록체인 플랫폼으로, 다양한 코인과 스왑 기능을 내장하여 기존 코인의 유통에서 중앙집중화된 거래 솔루션을 탈피하고자 거래 탈중앙화를 기획하였습니다.

UBMS의 의미는 아래와 같습니다:

U - UBMSCOIN
B - Bitcoin
M - Matching
S - Swap
비트코인 이후 다양한 코인들이 수도 없이 만들어지고 유통되었지만, 영업력을 앞세워 홍보만 할 뿐 실제로 기능의 혁신을 추구한 코인은 극히 소수입니다. 거래의 탈중앙화를 해결하기 위한 우리의 노력은 2018년부터 치열하게 시도되었습니다. 수년간의 시행착오를 거쳐 백서를 작성하게 되었습니다.

UBMS 알고리즘은 추후 백서 갱신 시 소개하도록 하겠습니다.

2. 문제점
현재의 금융 시스템은 다음과 같은 문제점을 안고 있습니다:

높은 거래 수수료: 국제 송금 시 높은 비용 발생.
느린 거래 속도: 거래 완료까지 평균 10분 이상 소요.
중앙화된 통제: 단일 기관에 의한 거래 관리로 인한 투명성 부족.
제한된 접근성: 금융 서비스에 접근하기 어려운 지역이 많음.
3. 해결 방안
**UBMSCOIN (UBMS)**은(는) 다음과 같은 혁신적인 솔루션을 제공합니다:

저렴한 수수료: 효율적인 블록체인 기술을 통해 거래 수수료를 대폭 절감.
빠른 거래 속도: 지속적인 패치를 통해 거래 속도 향상.
운영 탈중앙화: 분산형 네트워크를 통해 중앙 기관의 개입 없이 운영.
거래 탈중앙화: 자체 스왑 시스템 내장 (UBMS-Matching-Swap).
글로벌 접근성: 인터넷만 연결되어 있으면 누구나 쉽게 이용 가능.

7. 결론
**UBMSCOIN (UBMS)**는 거래의 탈중앙화에 도전합니다. 다년간 여러 업체에 거래소 솔루션을 만들어 납품하면서 거래소의 장단점을 누구보다 잘 알고 있기에 많은 경험을 코인에 적용하였습니다. 저희가 아니라도 언젠가는 누군가가 개발할 코인이라 생각합니다.

백서의 새로운 버전이 갱신될 때는 채굴 관련 신기능에 대한 소개를 보강할 예정입니다. 2024년 늦은 감은 있지만, 이상적인 코인을 추구하는 개발자들의 염원과 노력이 귀하의 삶에 조금이나마 보탬이 되길 희망합니다.
